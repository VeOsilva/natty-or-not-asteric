 📒 Descrição
Natty or Not AI é um projeto voltado para a detecção e classificação de conteúdo gerado por humanos (natural) versus conteúdo gerado por inteligência artificial (artificial). 
Nosso objetivo é criar um conjunto de ferramentas e modelos que possam ser utilizados para identificar se um vídeo, podcast, livro ou qualquer outro tipo de mídia
foi criado por uma pessoa ou por uma IA.
